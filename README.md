# 🧪 Práctica de Ramas en Equipo

Este repositorio contiene la **primera práctica de trabajo con ramas en equipo**, donde se aplican conceptos básicos de control de versiones utilizando Git.

## 🎯 Objetivo

El objetivo de esta práctica es:

* Aprender a crear y gestionar ramas
* Trabajar de forma colaborativa en un mismo repositorio
* Realizar fusiones (*merge*) de cambios hacia la rama principal
